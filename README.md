#UNP_SQL_PROJECT_E_COMMERCE_STORE_DATASET_ANALYSIS

Description:
This project builds a complete E-Commerce Store Database (aaktastore) using SQL with structured tables for Products, Customers, Orders, Payments, and Inventory.
It establishes relational integrity using primary and foreign keys to connect customers, orders, and payment transactions.
The project performs business analysis including total revenue, pending revenue, monthly sales trends, customer spending ranking, and inventory valuation.
Advanced SQL queries such as JOINs, GROUP BY, aggregate functions, ranking, and trend analysis are used to generate insights.
This project demonstrates practical database design, relational modeling, and real-world business analytics using SQL.
